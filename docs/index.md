### 功能特性

+ 沉浸式。全屏状态下使用不影响游戏窗口
+ 图片识别。内置OCR模型，可识别单词和短语
+ 绿色轻量。免安装，包体仅13M

### 使用说明

将`playdict-x.x.zip`解压到你的文件夹，运行`playdict.exe`

+ F1键/鼠标中键：截屏划词
+ F2键：显示或隐藏面板
+ F3键：退出程序
